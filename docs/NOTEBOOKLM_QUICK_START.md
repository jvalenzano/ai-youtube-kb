# NotebookLM Quick Start Checklist

A condensed checklist for creating your public NotebookLM notebook.

## Pre-Flight Checklist

- [ ] Content staged in `notebooks/notebooklm-staging/`
- [ ] All files properly named with video IDs
- [ ] Metadata embedded in companion files
- [ ] Content reviewed and finalized

## Step-by-Step (15 minutes)

### 1. Create Notebook (2 min)
- [ ] Go to https://notebooklm.google.com
- [ ] Sign in with Google account (personal, not Workspace)
- [ ] Click "New Notebook"
- [ ] Name: "AI Agents: Complete Learning Guide"
- [ ] Type: "Learning Notebook"

### 2. Upload Sources (5 min)
- [ ] Open `notebooks/notebooklm-staging/` folder
- [ ] Upload transcript files first (`*_transcript_*.txt`)
- [ ] Upload slide images (`*_slide_*.png`)
- [ ] Upload companion files (`*_slide_*.txt`)
- [ ] Wait for processing to complete
